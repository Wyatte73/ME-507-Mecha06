Detailed Description

Header file for the parallel LCD driver.

This file defines the LCD_param_t structure and the public functions used to initialize and control an LCD using GPIO pins on an STM32 microcontroller.

This function sends the LCD clear-display command. The command requires a longer delay than most LCD commands, so a delay is included before the data lines are cleared.

◆ Init()

void Init ( LCD_param_t inp)

Initializes the LCD.

This function performs the startup sequence for an HD44780-style LCD operating in 8-bit mode. It clears the data lines, waits for the LCD to power up, configures the display mode, clears the display, sets the entry mode, and turns the display on.

Parameters
inpLCD pin configuration structure.

This function performs the standard HD44780 initialization sequence for 8-bit mode. It configures the display for 8-bit data, two display lines, and a 5x8 font. It then clears the display, sets the cursor entry mode, and turns the display on with the cursor and blinking disabled.

Parameters
inpLCD pin configuration structure.
