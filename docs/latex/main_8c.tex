\doxysubsection{Detailed Description}
:\+ Main program body. 

This program initializes the STM32 peripherals, configures an SPI-\/controlled motor driver, initializes an LCD, measures an input pulse width using timer input capture, converts that measurement into a distance value, displays the distance on the LCD, and updates the motor target velocity over SPI.

Timer input-\/capture callback used to measure input pulse width. 

This callback is called automatically by the HAL when an input-\/capture event occurs. It measures the time between a rising edge and a falling edge on TIM2 channel 1.

The function first captures the rising-\/edge timer value, then switches the channel polarity to capture the falling edge. After the falling edge is captured, it calculates the pulse width and switches back to rising-\/edge capture for the next pulse.
